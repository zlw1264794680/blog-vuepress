# 安装tips

npm init

npm i vant-weapp -S --production

如果构建npm失败，可能是npm初始化安装的目录不对。

终端切换到，红框指定的目录，再进行npm初始化、安装、构建

# 其他步骤

按照官网步骤安装、构建npm即可。
