# 前提步骤

  • 手机处于开发者模式（MIUI系统路径：设置、我的设备、全部参数、MUUI版本【连续点击，直到提示处于开发者】）
  • 手机开启usb调试，允许usb安装（MIUI系统路径：设置、更多设置、开发者选项、usb调试）

# tips

确保数据线，能够传输文件！！！

# 操作步骤

usb链接手机和电脑，手机并未提示usb配置选择，很有可能是数据线仅支持充电导致的！！！

手机提示usb配置选择，选择文件传输。

执行 npm run android、npm run start即可。

# 构建app，自动到AVD

#查看下当前存在的模拟器、真机设备
adb devices

# 结果显示
List of devices attached
47bffad9 unauthorized

执行 adb devices 命令后真机设备显示 unauthorized 设备未授权。

# 原因解析

之前调试时设备授权过，授权后的手机不会再提示授权，但此次调试需要重新授权

# 解决方案

获取设备VID

USB 将手机与电脑连接，windows右击电脑属性→设备管理器→便携设备→属性→详细信息→硬件id，如下设备VID为05C6

配置adb_usb.ini

在电脑用户文件夹下找到 C:\Users\Administrator\.android 文件夹，查看文件夹下是否有 adb_usb.ini 这个文件，若没有则新建一个txt文件修改后缀，输入以下内容并替换设备VID的值即可。

# ANDROID 3RD PARTY USB VENDOR ID LIST -- DO NOT EDIT.
# USE 'android update adb' TO GENERATE.
# 1 USB VENDOR ID PER LINE.
0x05C6

配置完成后，执行命令 adb kill-server，adb start-server

手机重新连接电脑后就会弹出设备授权调试提示了，确认授权。

重新执行命令 adb devices 如下，设备调试连接成功。

#查看下当前存在的模拟器、真机设备
adb devices

# 结果显示
List of devices attached
47bffad9 device
