夜神模拟器打开后，执行 npm run android、npm run start即可。

# 问题：构建app，自动到AVD

执行命令后，构建app，自动到AVD，而不是夜神，如何解决？

#查看下当前存在的模拟器、真机设备
adb devices

结果如下：

List of devices attached

显示没有模拟器存在，但是我已经开启了夜神了。

解决方法：

  • 将 SDK 目录下的 platform-tools 下的 adb.exe，复制到 夜神安装目录 Nox\bin 下，进行替换
  • 将 adb.exe 复制出来，重命名为 nox_adb.exe，也复制到 夜神安装目录 Nox\bin 下，进行替换
  • 重启模拟器就可以了。
# 再次执行查看模拟器
adb devices

# 结果显示
List of devices attached
127.0.0.1:62001 device

链接夜神模拟器：

adb connect 127.0.0.1:62001

再次执行 npm run android、npm run start即可。
