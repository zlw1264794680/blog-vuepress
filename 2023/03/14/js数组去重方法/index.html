JS数组去重方法

zlw
2023-03-14
  • JS

# 推荐

# 利用 Set 和 Array.from/展开运算符（...）

Set对象：是值的集合，你可以按照插入的顺序迭代它的元素。 Set中的元素只会出现一次，即Set中的元素是唯一的。

在 Set 对象中：

  • 认为 +0 和 -0 是相等的
  • 认为 NaN 是相等的

Array.from() 方法：对一个类似数组或可迭代对象创建一个新的，浅拷贝的数组实例。

const arr = [1, 1, 'a', undefined, NaN, null, '1', 'A', undefined, NaN, null, -0, +0]

function duplicateRemoval(arr) {
    const s = new Set(arr)
    return Array.from(s)
    // return [...s] // 也可以使用展开运算符
}

console.log(duplicateRemoval(arr)); // [1, 'a', undefined, NaN, null, '1', 'A', 0]

# 利用 Map 对象

Map 对象是 JavaScript 提供的一种数据结构，结构为键值对形式，将数组元素作为 map 的键存入，然后结合 has() 和 set() 方法判断键是否重复。

Map 对象：用于保存键值对，并且能够记住键的原始插入顺序。任何值（对象或者原始值）都可以作为一个键或一个值。

const arr = [1, 1, 'a', undefined, NaN, null, '1', 'A', undefined, NaN, null, -0, +0]

function duplicateRemoval(arr) {
    const m = new Map()
    const newArr = []
    arr.forEach(item => {
        if (!m.has(item)) {
            m.set(item, 1)
            newArr.push(item)
        }
    })
    return newArr
}

console.log(duplicateRemoval(arr)); // [1, 'a', undefined, NaN, null, '1', 'A', -0]

可以看到，利用Set 和 利用Map，都能实现数组去重，但是去重后的数组，Set会对+0和-0进行处理，返回显示0； 而Map，不会进行任何处理。

在 Map 对象中：

  • 认为 +0 和 -0 是相等的，即 has(0) === has(-0)
  • 认为 NaN 是相等的

# 利用对象

原理同,利用Map对象一样

const arr = [1, 1, 'a', undefined, NaN, null, '1', 'A', undefined, NaN, null, -0, +0]

function duplicateRemoval(arr) {
    const m = {}
    const newArr = []
    arr.forEach(item => {
        if (!m[item]) {
            m[item] = 1
            newArr.push(item)
        }
    })
    return newArr
}

console.log(duplicateRemoval(arr)); // [1, 'a', undefined, NaN, null, 'A', -0]

可以看出，与利用Map处理的区别，利用对象去重，少了一个1。

原因在于：对象中，键只能接受字符串和symbol，所以m[1] = 1 和 m['1'] = 1的处理上，键都是字符串1，这就是他少了一个1的原因。

# 不推荐

对于所有数组去重方式，涉及到双循环的都不是很推荐。因为在数组很大的情况下，性能损耗大。时间复杂度为O（n）。

推荐
利用 Set 和 Array.from/展开运算符（...）
利用 Map 对象
利用对象
不推荐
